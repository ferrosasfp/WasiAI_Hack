# Libraries

Librerías y utilidades core del proyecto (EVM-only / Avalanche).

## Módulos

- **cache.ts** - Cache utilities con TTL
- **crypto.ts** - Funciones criptográficas (hashing, etc.)
- **db.ts** - Cliente PostgreSQL/Neon para queries
- **draft-utils.ts** - Utilidades para drafts del wizard
- **fetchEvmModel.ts** - Fetch de modelos EVM con metadata IPFS
- **indexer.ts** - Indexador de modelos desde blockchain a DB
- **indexer-single.ts** - Indexador de modelo individual
- **metrics.ts** - Métricas y contadores
- **prefetch.ts** - Prefetch de datos para SSR
